# Sistem Manajemen Produksi (Tugas 6)

Aplikasi Command Line Interface (CLI) berbasis Python dan MongoDB untuk mengelola, menganalisis, dan mengekspor data produksi mesin.

## Persyaratan Sistem
1. Python 3.7 atau lebih baru terinstal.
2. MongoDB Server (lokal/Compass) sedang berjalan.
3. Library Python: `pymongo`, `pandas`, `python-dotenv`.

## Cara Instalasi
1. Buka terminal pada folder proyek ini.
2. Instal library yang dibutuhkan dengan menjalankan:
   ```bash
   pip install pymongo pandas python-dotenv
